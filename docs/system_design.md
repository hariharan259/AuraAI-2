## 2. State Management Architecture
AuraAI uses React's Context API through `AuraContext` to manage application state across pages.

### AuraContext State Properties:
* `state.analysisResult`: The most recent report generated by the multi-agent system.
* `state.analysisStatus`: `'idle' | 'running' | 'complete'`.
* `state.agentStatus`: An object tracking the progress of individual agents:
  ```json
  {
    "dermatologist": "idle" | "running" | "complete",
    "trichologist": "idle" | "running" | "complete",
    "ingredientScientist": "idle" | "running" | "complete",
    "beautyCoach": "idle" | "running" | "complete",
    "outcomePredictor": "idle" | "running" | "complete"
  }
  ```
* `state.history`: A ledger of all past reports, allowing users to load and restore previous diagnostics.

## 3. UI Design System & Styling Specification

AuraAI features a dark, glassmorphic visual system constructed entirely with Vanilla CSS in `src/index.css`.

### Custom Color Palette (HSL & Hex CSS Variables)
```css
:root {
  --font-sans: 'Inter', system-ui, sans-serif;
  --font-display: 'Playfair Display', serif;

  /* Theme Colors */
  --bg-primary: #0F0E13;
  --bg-secondary: #16141D;
  --bg-tertiary: #1E1A27;

  /* Typography Colors */
  --text-primary: #FFFFFF;
  --text-secondary: #A7A1B8;
  --text-muted: #6E697F;

  /* Aura Accent Glows */
  --aura-primary: #8B5CF6;
  --aura-primary-light: #A78BFA;
  --aura-primary-glow: rgba(139, 92, 246, 0.15);

  --aura-secondary: #EC4899;
  --aura-secondary-light: #F472B6;
  --aura-secondary-glow: rgba(236, 72, 153, 0.15);

  --aura-gold: #F59E0B;
  --aura-gold-light: #FBBF24;
  --aura-gold-glow: rgba(245, 158, 11, 0.15);

  --aura-green: #10B981;
  --aura-green-light: #34D399;
  --aura-green-glow: rgba(16, 185, 129, 0.15);

  /* Glassmorphism Definitions */
  --glass-bg: rgba(22, 20, 29, 0.65);
  --glass-bg-strong: rgba(30, 26, 39, 0.85);
  --glass-border: rgba(255, 255, 255, 0.05);
  --glass-border-hover: rgba(255, 255, 255, 0.12);
  --glass-shadow: 0 8px 32px 0 rgba(0, 0, 0, 0.37);
}
```

### Visual Styling Guidelines:
1. **Glassmorphism**: Cards and dashboards must implement `background: var(--glass-bg)`, `backdrop-filter: blur(16px)`, and `border: 1px solid var(--glass-border)`.
2. **Typography**: Headings use the premium serif family `var(--font-display)` (Playfair Display) to align with luxury beauty branding, while UI elements and body text use the clean geometric sans `var(--font-sans)` (Inter).
3. **Animations**: Transition durations should maintain a consistent `0.3s cubic-bezier(0.4, 0, 0.2, 1)` feel for responsive hover states and component appearances.
